四个设计要点

纯 CSS，零 JS
横向滚动
表格整体最小宽度 580px，窄屏时整张表横向滑动，不挤压、不换行。
首列吸顶
参数列（心率/收缩压/呼吸频率）滚动时始终贴在屏幕左边，永远看得见。
智能换行
word-break: keep-all + line-break: strict，让"呼吸频率"自动换到"（次/min）"，不在括号后断、不单字成行。
内联提示
提示放在表头上方（不浮层），用户横向滑动一次后自动隐藏，不再骚扰。sessionStorage 记录"已见过"。

PC 端不受影响

只在 ≤768px 启用，桌面端完全保持原样

改造范围：styles.css + app.js（约 20 行 JS：自动给表格包滚动容器 + 加内联提示）。不动 HTML、不动后端。所有 docx/rtf/txt 文档里的表格自动受益。
工作量：~30 行 CSS + ~20 行 JS，15 分钟。
兼容性：sticky/overflow/line-break 是 2017 年后的标准，主流浏览器全部支持。
提示的"自动隐藏"机制：表格渲染后检测 scrollWidth > clientWidth，需要滚动才显示提示；用户第一次滑动表格后即隐藏，sessionStorage 记录"已见过"。
